# Image CDN Collection

A curated collection of publicly available images served via [jsDelivr CDN](https://www.jsdelivr.com/) for high-speed global delivery.

## Usage

Access images via jsDelivr:

```
https://cdn.jsdelivr.net/gh/mss-col/img-cdn/[path]/[filename]
```

**Example:**
```
https://cdn.jsdelivr.net/gh/mss-col/img-cdn/posters/example.jpg
```

## Directory Structure

```
img-cdn/
├── channels/    # Channel logos
├── posters/     # Poster images
├── brands/      # Brand logos
└── icons/       # Icon assets
```
